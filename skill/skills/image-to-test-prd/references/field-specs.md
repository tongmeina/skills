# 常用字段规格参考

本文档提供常见输入字段的规格参考，用于PRD编写时的快速查阅。

---

## 1. 身份信息类

| 字段名 | 数据类型 | 长度/范围 | 格式/正则 | 说明 |
|--------|----------|-----------|-----------|------|
| 手机号 | String | 11位 | `^1[3-9]\d{9}$` | 中国大陆手机号 |
| 身份证号 | String | 18位 | `^[1-9]\d{5}(19|20)\d{2}(0[1-9]|1[0-2])(0[1-9]|[12]\d|3[01])\d{3}[\dXx]$` | 二代身份证 |
| 邮箱 | String | 5-100字符 | `^[\w.-]+@[\w.-]+\.\w{2,}$` | 标准邮箱格式 |
| 用户名 | String | 6-20字符 | `^[a-zA-Z0-9_]+$` | 字母数字下划线 |
| 密码 | String | 8-20字符 | 含大小写字母和数字 | 强密码要求 |
| 真实姓名 | String | 2-20字符 | `^[\u4e00-\u9fa5]{2,20}$` | 中文姓名 |
| 昵称 | String | 2-20字符 | 中英文数字 | 允许特殊字符 |

## 2. 编号/标识类

| 字段名 | 数据类型 | 长度/范围 | 格式/正则 | 说明 |
|--------|----------|-----------|-----------|------|
| 卡号 | String | 6-20位 | `^[0-9]{6,20}$` | 纯数字 |
| 设备ID | String | 1-32位 | `^[A-Za-z0-9]+$` | 字母数字 |
| 订单号 | String | 16-32位 | `^[A-Za-z0-9-]+$` | 系统生成 |
| 流水号 | String | 20-36位 | `^[A-Za-z0-9]+$` | 系统生成 |
| UUID | String | 36位 | `^[0-9a-f]{8}-[0-9a-f]{4}-[0-9a-f]{4}-[0-9a-f]{4}-[0-9a-f]{12}$` | 标准UUID |
| 工号 | String | 6-10位 | `^[A-Z0-9]+$` | 大写字母数字 |

## 9. 边界值测试点模板

### 字符串长度边界（以6-20为例）

| 测试点 | 输入值 | 预期结果 |
|--------|--------|----------|
| 离点(下) | 5字符 "abcde" | 失败，提示长度不足 |
| 上点(下) | 6字符 "abcdef" | 成功 |
| 内点 | 13字符 "test_user_123" | 成功 |
| 上点(上) | 20字符 "test_user_1234567890" | 成功 |
| 离点(上) | 21字符 "test_user_12345678901" | 失败，提示长度超出 |
| 空值 | "" | 失败，提示不能为空（如必填） |

### 数值范围边界（以0.01-99999.99为例）

| 测试点 | 输入值 | 预期结果 |
|--------|--------|----------|
| 离点(下) | 0 | 失败，提示金额不能为0 |
| 上点(下) | 0.01 | 成功 |
| 内点 | 5000.00 | 成功 |
| 上点(上) | 99999.99 | 成功 |
| 离点(上) | 100000.00 | 失败，提示金额超出范围 |
| 负数 | -1 | 失败，提示金额不能为负 |
| 精度超出 | 0.001 | 四舍五入或失败 |
